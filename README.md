## Overview
Data analysis project focused on cleaning the Blinkit sales dataset, deriving KPIs, and visualizing key sales drivers.

## Analysis Highlights
* **Data Cleaning:** Handled missing values and data types using **Pandas** and **NumPy**.
* **KPIs Derived:** Total Sales, Average Sales, Item Volume, and Average Ratings.
* **Key Drivers Visualized:** Sales breakdown by **Fat Content**, **Item Type**, **Outlet Size**, **Location Type**, and **Establishment Year**.
* **Visualization:** Used **Matplotlib** and **Seaborn** to create Bar, Pie, and Line charts.

## Tech Stack
* Python (Jupyter Notebook)
* Pandas, NumPy
* Matplotlib, Seaborn
